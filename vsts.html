BooBass

This VST is a great sounding guitar. It has 3 simple knobs that you can adjust to make your guitar sound however you would like it to.

Sytrus

Sytrus is a great synth to use. If you are into EDM or just would like to add a synth to your song, this is a great tool to use

Sakura

Sakura is a Japaneese style synth. This is great for soft sounding melodies, and can be used for almost any style of song.
